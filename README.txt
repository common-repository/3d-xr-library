== Description ==
 
 Amazing and easy to use 3D WordPress plugin. 
 
= Want to know how this plugin is beneficial? Here are some of its features: =
 
* A myriad of models: Allows you to add various 3D models and view it. 

* Trouble free visibility: Accessible by just adding a short codeusing shortcode [3dmodelviewer] and you are good to go. 

* Display anywhere: Displays 3D models on WordPress page, post or any custom post type. 

* Supported formats: Displayed in the gltf and glb formats. 

* AR support: Supports Augmented Reality, I.e., you will be able to experience it by placing in the real world by visiting the URL from your mobile browser.

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go thorugh Settings/Permalinks and add new slug to replace wp-admin.


== Frequently Asked Questions ==

=  How can I use this plugin? =  

1. Active plugin and you will see 3d model in backend to add 3d model data.

2. Use this shortcode [3dmodelviewer] in backend of page or post to display 3d model in front side.
